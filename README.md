# 📝 devnote

`devnote` is a lightweight, fast, and minimal **Single Page Application (SPA)** built using the **MERN Stack** (MongoDB, Express.js, React, Node.js). It is designed specifically for developers and students to quickly jot down thoughts, structure ideas, or organize daily tasks without page-reload friction.

---

## 🚀 Features

- **Single Page Architecture (SPA):** Instant actions, routing, and updates without full page refreshes.
- **Rich Dashboard:** Clean layout to create, view, edit, and delete notes in real time.
- **Dynamic Search:** Filter through your saved notes instantly as you type.
- **RESTful Backend API:** Fully backed by a robust Node.js and Express server with MongoDB Atlas database integration.
- **Responsive Design:** Completely optimized for both desktop and mobile screens.

---

## 🛠️ Tech Stack

- **Frontend:** React.js, HTML5, CSS3 / Bootstrap (or Tailwind CSS)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (via Mongoose)
- **State Management & Routing:** React Router DOM, Axios
